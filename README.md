运行依赖

Python 3.x（在 Windows 上）
建议安装：pystray、Pillow
安装命令：pip install pystray pillow
如何打包为 .exe（使用 PyInstaller）

安装 PyInstaller:
pip install pyinstaller
（可选）准备一个图标文件 app.ico（如果需要 .exe 显示自定义图标）。

执行： pyinstaller --clean --onefile --windowed your_script.py
